#Influence

They are the folks driving conversations

The size of circles represents how they are influential

What’s influence?

Influence on Twittersphere can be elusive to define due to the limitation of the Twitter API. Our evaluation method is like this.

Each time you type a hashtag, the API returns the most popular 100 tweets around the # based on their algorithm. We sort them out on the basis of the number of retweets and display the top 5 results as bigger circles. But the smaller circles within each bigger circle are calculated by their follower numbers.

We acknowledge that our method is far from coherent as for the way of evaluating influence. It’s basically because the Twitter API doesn't track retweets of a retweet. Also, let us add a note of caution, the precision of the number of retweets is somewhat elusive again, as the API doesn’t sum up the number of retweets and edited retweets.
